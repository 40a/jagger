/// 1. System resources (collected with use of [Sigar](http://www.hyperic.com/products/sigar))
///   * CPU
///   * Memory
///   * Network
///   * Disk
/// 2. JVM resources (collected via JMX)
///   * Heap memory and non heap memory
///   * Minor and major GC runs
///   * Thread count
